## 🏁 Getting Started <a name = "getting_started"></a>

### Prerequisites


Python installed in version 3.12.3

```bash
# python --version
Python 3.12.3
```

### Installing


Installing the libraries from the "requirements.txt" file

```bash
pip install -r requirements.txt
```
